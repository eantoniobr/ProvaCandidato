# Solução - Prova Candidato.

- Explique o funcionamento dos blocos try{}, catch{} e finally{}.

Os blocos servem para tratamentos de exceções.

O try vai tentar executar uma lista de instruções.
O catch vai ser executado caso uma exceção for encontrada.
O finally será executado independente se existir ou não uma exceção.

-  Qual a função do Migrations no EntityFramework e qual problema ele tem a intenção de  resolver?

O Migration tem a intenção de nos ajudar a realizar atualizações no esquema e estrutura de tabelas do banco de dados sem precisarmos descartar ou recriar o banco de dados.

- Qual o benefício que a ViewBag traz sendo do tipo dynamic?

O benefício é que podemos utilizar qualquer tipo de dado para uso em uma View.
